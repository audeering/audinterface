audinterface.utils¶

read_audio¶

audinterface.utils.read_audio(path, start=None, end=None)[source]¶

Reads (segment of an) audio file.

Parameters

  • path (str) – path to audio file

  • start (Optional[Timedelta]) – read from this position

  • end (Optional[Timedelta]) – read until this position

Returns

array with signal values in shape (channels, samples) sampling_rate: sampling rate in Hz

Return type

signal
